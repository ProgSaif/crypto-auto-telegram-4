# 📈 Crypto Auto Signal Telegram Bot
An automated **crypto trading signal bot** that scans Binance market data and sends **LONG / SHORT trade signals** to a Telegram channel.
The bot continuously monitors USDT trading pairs and posts signals based on **price movement, trend detection, and confidence scoring**.



# 🚀 Features
✅ Automatic crypto signal detection
✅ Binance market data scanning
✅ LONG and SHORT trade setups
✅ Auto TP / SL calculation
✅ Confidence score for every signal
✅ Telegram channel posting
✅ Auto-delete messages after a time period
✅ Continuous market scanning
